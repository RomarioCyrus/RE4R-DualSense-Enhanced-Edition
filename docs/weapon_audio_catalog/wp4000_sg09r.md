| `event_0258` / Wwise ID `1972347088` | Reload finish | Re-confirmed live (2025-06-27) after a 2025-06-27 capture caught it firing right after the magazine-insert tick; the earlier "no separate finish cue" conclusion is superseded |

The insertion cue was perceptibly late before weapon/ammo polling was tightened
from six to three update frames. The latest in-game pass confirmed the reduced
latency mapping feels good. `event_0271` (the original insert candidate) never
fired in any live `postRequestInfo` capture and was removed from the runtime
mapping; ammo-delta polling for insert was also replaced by the direct Wwise
gate to fix a missing-insert-sound edge case (firing the externally-chambered
extra round when the magazine is otherwise full).

The confirmed runtime mapping takes precedence over waveform-only
classification. Events `0232–0252` may be granular component layers, handling
events, or alternate animation states, but they have not yet been synchronized
with gameplay.

## Event-group and deduplication notes

- Source set: 55 WAV files, 53 unique decoded WAV contents.
- `event_0234`: three WEM references, but two decode to identical WAV content
  (`344270339` and `329003081`).
- `event_0250`: three WEM references, but two decode to identical WAV content
  (`970245485` and `647324257`).
- `event_0254` contains material-switched branches rather than one ordinary
  round-robin set.
- `event_0260`: four variants.
- `event_0271`: three variants.

## Current runtime mapping

- `wp4000_reload_start*` contains the four `event_0260` variants.
- `wp4000_reload_insert*` contains the three `event_0271` variants.
- `wp4000_reload_finish*` files still exist in the source sound directory and
  originate from `event_0258`, but the SG-09 R profile does not emit a finish
  event.

## Comparison result

The supplied analysis is useful for selecting future high-detail speaker and
haptic layers:

- strongest speaker/mechanical candidate: `event_0248`;
- strongest magazine confirmation candidate: `event_0240`;
- useful slide-tension candidate: `event_0244`.

It does not invalidate the current reload implementation. `event_0260` and
`event_0271` remain the only gameplay-confirmed reload groups. `event_0260`
also has a separately confirmed empty-trigger/dry-fire route when detected by
runtime Wwise event ID and gated by `ammo=0`. The proposed
`Fire` classification for `event_0252` is not accepted without direct in-game
comparison.
