## Runtime mapping

- Per-shell reload insertion is implemented using the `event_0213` /
  `event_0215` candidate family.
- Pump-open and pump-close are normally emitted after an actual ammunition
  decrease.
- The pump cycle is scheduled roughly one second after the shot only when
  ammunition remains. If the shot empties the weapon, the immediate cycle is
  suppressed and deferred until the following reload ends.
- Shell-drop, safety, lifter, and dry-fire events are not implemented.
